## 修士論文要旨のテンプレート 
2024年度分\
元のテンプレートから卒業論文予稿のテンプレートのものに移行し，タイトルを同じ見た目に修正。

### 修正した理由
- 元のテンプレートでは，図を挿入しようとしたときに表の裏に図が隠れて画像が挿入できない、または、挿入できても文章内の上下に寄せることができないという問題があった。
  - ２段組を「documentclass」内で呼び出して全体を２段組構成にするのではなく，全体は１段組構成で、タイトルの表より下の部分を２段組構成環境にして、その中に本文を記載するものであったため
- 卒業論文のテンプレートでは，２段組の各１列が各ページ的な扱いで利用できた。
- 元のテンプレートは，タイトルなどの設定をすべて「main.tex」内に記載していた。

### 修正点
- テンプレートを卒業論文のものに変更し，「proceeding_bachelor.sty」(後ほど名前は修正) ファイル内のタイトルのところを修士論文予稿のテンプレートの表の部分に変更
- 本文の外側の空白は卒業論文よりサイドが広いため，数値を12mmから20mm(上下の空白と同じ)に変更
